Summary: Cloudiness vs. Humidity

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

The third relationship investigated was that between cloudiness and latitude. It was assumed that there was no relationship between cloudiness and latitude. This assumption proved to be true.

Moreover, the correlation was tested for each hemisphere separately, to see if the initial assumption would hold up when only looking at a hemisphere individually. The data shows that the assumption is also true here as well, with the linear regression equations possessing R^2 values of 0.03 for the Northern Hemisphere and 0.07 for the Southern Hemisphere. This would suggest that there are other factors that affect cloudiness much more strongly - although it's not so apparent what those would be.
